# Password Strength Analyzer & Advanced Wordlist Generator

A Python tool that analyzes password strength with realistic entropy, crack time, and real-world patterns. Generates custom wordlists using personal details, leetspeak, common passwords, and case mutations. Equipped with both CLI and GUI (Tkinter) interfaces.

## Features

- Real-world password analysis (entropy, crack time, warnings, suggestions)
- Custom wordlist generation using personal info (+leetspeak, casing, suffixes, reversals)
- Incorporates latest common password leaks
- CLI and GUI (Tkinter) interfaces
